# Part 2: Cấu trúc Dữ liệu & Thuật toán (Data Structures & Algorithms - DSA)

Chào mừng bạn đến với **Module 2: Cấu trúc Dữ liệu & Thuật toán**.

Tài liệu này tiếp nối kiến thức nền tảng từ Module 1, tập trung vào việc phân tích các cơ chế lưu trữ dữ liệu (Data Structures) và các phương pháp xử lý dữ liệu (Algorithms) ở cấp độ hệ thống. Mục tiêu cốt lõi là đánh giá sự đánh đổi (Trade-offs) giữa hiệu suất xử lý CPU (Thời gian) và dung lượng RAM (Không gian), từ đó xây dựng tư duy thuật toán nhằm tối ưu hóa các ứng dụng thực tiễn.

Các bài học được viết theo chuẩn mực khoa học, kết hợp phân tích rễ sâu và hệ thống minh họa sơ đồ luồng dữ liệu (Mermaid Diagrams).
